---
title: 'Transaction API'
description: 'Process card and ACH transactions through the AuxVault platform'
---

The `/transaction` endpoint allows merchants to process **Card** or **ACH** transactions using AUXVAULT's public API. The API supports one-time sale or authorization-based payments and can optionally tokenize payment methods for future use.

<Endpoint method="POST" url="https://dev-api.auxvault.net/api/v1/public/transaction">

<ParamField type="header" name="Authorization" required>
  Your API key for authentication.
</ParamField>

<ParamField body="Amount" type="decimal" required>
  The amount of the transaction.
</ParamField>

<ParamField body="BillingAddress" type="string">
  The billing address.
</ParamField>

<ParamField body="BillingCity" type="string">
  The billing city.
</ParamField>

<ParamField body="BillingCountry" type="integer" required>
  The billing country.
</ParamField>

<ParamField body="BillingCountryCode" type="string" required>
  The billing country code.
</ParamField>

<ParamField body="BillingCustomerName" type="string" required>
  The name of the billing customer.
</ParamField>

<ParamField body="BillingEmail" type="string" required>
  The billing email address.
</ParamField>

<ParamField body="BillingPhoneNumber" type="string" required>
  The billing phone number.
</ParamField>

<ParamField body="BillingPostalCode" type="string" required>
  The billing postal code.
</ParamField>

<ParamField body="BillingState" type="integer">
  The billing state.
</ParamField>

<ParamField body="ShippingSameAsBilling" type="boolean">
  Set to `true` if shipping address is the same as billing address.
</ParamField>

<ParamField body="ShippingCustomerName" type="string">
  The name of the shipping customer.
</ParamField>

<ParamField body="ShippingEmail" type="string">
  Email address of the shipping customer.
</ParamField>

<ParamField body="ShippingPhoneNumber" type="string">
  Phone number of the shipping customer.
</ParamField>

<ParamField body="ShippingAddress" type="string">
  The shipping address.
</ParamField>

<ParamField body="ShippingCountry" type="string">
  The shipping country.
</ParamField>

<ParamField body="ShippingCity" type="string">
  The shipping city.
</ParamField>

<ParamField body="ShippingState" type="string">
  The shipping state.
</ParamField>

<ParamField body="ShippingPostalCode" type="string">
  The shipping postal code.
</ParamField>

<ParamField body="CardNumber" type="string" required>
  The card number.
</ParamField>

<ParamField body="Cvv" type="string" required>
  The card verification value (CVV).
</ParamField>

<ParamField body="ExpiryDate" type="string" required>
  The card expiry date in MM/YY format.
</ParamField>

<ParamField body="SuggestedMode" type="string" required>
  The suggested payment mode (e.g., "Card").
</ParamField>

<ParamField body="ConvenienceFeeActive" type="boolean" required>
  Set to `true` to activate the convenience fee.
</ParamField>

<ParamField body="TransactionType" type="string">
  `1` for Sale, `2` for Authorization.
</ParamField>

<ParamField body="PaymentTokenization" type="boolean">
  Set to `true` to tokenize the payment method for future use.
</ParamField>

<ParamField body="IpAddress" type="string" required>
  IPv4 or IPv6 address of the end user.
</ParamField>

</Endpoint>

## ACH Transaction

Currently, we do not share ACH transaction details via this API. If you require ACH transaction data, please contact our support team.
