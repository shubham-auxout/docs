# Create Customer

This endpoint allows you to create a new customer profile in the system.

## API Endpoint
`https://dev-api.auxvault.net/api/v1/public/customer`

## Request Parameters
The request must be a POST request with the following parameters:

| Parameter   | Data Type | Required | Description                    |
|-------------|-----------|----------|--------------------------------|
| CustomerName| String    | Yes      | Full name of the customer      |
| Email       | String    | Yes      | Email address of the customer  |
| PhoneNumber | String    | Yes      | Phone number of the customer   |
| Address     | String    | No       | Street address of the customer |
| City        | String    | No       | City of the customer           |
| PostalCode  | String    | No       | Postal code of the customer    |
| StateId     | Integer   | No       | State ID of the customer       |
| CountryId   | Integer   | Yes      | Country ID of the customer     |
| CountryCode | String    | Yes      | Country code (e.g., +1 for US) |

## Example Request
```json
{
  "CustomerName": "John Doe",
  "Email": "john.doe@example.com",
  "PhoneNumber": "1234567890",
  "Address": "123 Main St",
  "City": "New York",
  "PostalCode": "10001",
  "StateId": 1,
  "CountryId": 1,
  "CountryCode": "+1"
}
```

## Response
The response will contain the status, message, and data of the transaction.

| Parameter | Data Type | Description                              |
|-----------|-----------|------------------------------------------|
| status    | String    | The status of the transaction            |
| message   | String    | A message describing the transaction status |
| data      | Object    | An object containing details of the transaction |

**Example Response**:
```json
{
  "status_code": 200,
  "message": "Customer created successfully",
  "data": {
    "CustomerId": "fe8bd3d5-9ebd-4722-bf44-23d88c250b34",
    "CustomerName": "John Doe",
    "Email": "john.doe@example.com",
    "PhoneNumber": "1234567890",
    "Address": "123 Main St",
    "City": "New York",
    "PostalCode": "10001",
    "StateId": 1,
    "CountryId": 1,
    "CountryCode": "+1",
    "GatewayCustomerId": "d0h93vn0i47dbksokv70",
    "createdAt": "2025-05-13T00:27:43.000Z"
  }
}
```

## Error Codes
If an error occurs, the response will contain an error code and a message.

| Error Code | Description               |
|------------|---------------------------|
| 400        | Bad Request               |
| 401        | Unauthorized              |
| 403        | Forbidden                 |
| 500        | Internal Server Error     |

## Security
To access this API endpoint, you will need to provide an API key in the header:

```javascript
var request = { 
  method: "post", 
  url: "https://dev-api.auxvault.net/api/v1/public/customer", 
  headers: { 
    "Content-Type": "application/json", 
    "authorization": "APIKEY"
  },
  data: JSON.stringify({
    "CustomerName": "John Doe",
    "Email": "john.doe@example.com",
    "PhoneNumber": "1234567890",
    "CountryId": 1,
    "CountryCode": "+1"
  })
}
```

# Customer List

This endpoint retrieves a list of all customers associated with your account.

## API Endpoint
`https://dev-api.auxvault.net/api/v1/public/customers`

## Request Parameters
The request must be a POST request with the following parameters:

| Parameter | Data Type         | Required | Description          |
|-----------|-------------------|----------|----------------------|
| sort      | String (asc / desc) | No       | for sorting the order |
| page      | Integer           | No       | For page number      |
| perPage   | Integer           | No       | For data count       |
| search    | String            | No       | For search           |

## Example Request
```json
{
  "req.query": "sort=\"\"&page=\"\"&perPage=\"\"&search=\"\""
}
```

## Response
The response will contain the status, message, and data of the transaction.

| Parameter | Data Type | Description                              |
|-----------|-----------|------------------------------------------|
| status    | String    | The status of the transaction            |
| message   | String    | A message describing the transaction status |
| data      | Object    | An object containing details of the transaction |

**Example Response**:
```json
{
  "status_code": 200,
  "message": "Customer found successfully",
  "data": [
    {
      "CustomerId": "fe8bd3d5-9ebd-4722-bf44-23d88c250b34",
      "CustomerName": "xxxxxxxxxxxxx",
      "Address": "",
      "City": "",
      "PostalCode": "54321",
      "StateId": null,
      "CountryId": 1,
      "PhoneNumber": "8976732233",
      "Email": "xxxxxxxxxxxx@gmail.com",
      "GatewayCustomerId": "d0h93vn0i47dbksokv70",
      "CountryCode": "+1",
      "createdAt": "2025-05-13T00:27:43.000Z"
    }
  ],
  "paging": {
    "total": 1,
    "pages": 1
  }
}
```

## Error Codes
If an error occurs, the response will contain an error code and a message.

| Error Code | Description               |
|------------|---------------------------|
| 400        | Bad Request               |
| 401        | Unauthorized              |
| 403        | Forbidden                 |
| 404        | Not Found                 |
| 500        | Internal Server Error     |

## Security
To access this API endpoint, you will need to provide an API key in the header:

```javascript
var request = { 
  method: "get", 
  url: "https://dev-api.auxvault.net/api/v1/public/customers?params", 
  headers: { 
    "Content-Type": "application/json", 
    "authorization": "APIKEY"
  }
}
```

# Customer Details

This endpoint allows you to view or update details for a specific customer, identified by their unique customer ID.

## API Endpoint
`https://dev-api.auxvault.net/api/v1/public/customer/fe8bd3d5-9ebd-4722-bf44-23d88c250b34 [id]`

## Request Parameters
The request must be a GET request with the following parameters:

| Parameter | Data Type | Required | Description                    |
|-----------|-----------|----------|--------------------------------|
| id        | String    | Yes      | For individual customer data   |

## Example Request
```json
{
  "req.params.id": "CustomerId from the user list"
```
