This section provides endpoints for creating, retrieving, and managing customer profiles in the AuxVault system.

## Create Customer

This endpoint allows you to create a new customer profile.

<Endpoint method="POST" url="https://dev-api.auxvault.net/api/v1/public/customer">

<ParamField type="header" name="Authorization" required>
  Your API key for authentication.
</ParamField>

<ParamField body="CustomerName" type="string" required>
  The full name of the customer.
</ParamField>

<ParamField body="Email" type="string" required>
  The email address of the customer.
</ParamField>

<ParamField body="PhoneNumber" type="string" required>
  The phone number of the customer.
</ParamField>

<ParamField body="CountryId" type="integer" required>
  The ID of the customer's country.
</ParamField>

<ParamField body="CountryCode" type="string" required>
  The country code for the phone number (e.g., `+1` for the US).
</ParamField>

<ParamField body="Address" type="string">
  The street address of the customer.
</ParamField>

<ParamField body="City" type="string">
  The city of the customer.
</ParamField>

<ParamField body="PostalCode" type="string">
  The postal code of the customer.
</ParamField>

<ParamField body="StateId" type="integer">
  The ID of the customer's state.
</ParamField>

</Endpoint>

### Example Request

```json
{
  "CustomerName": "John Doe",
  "Email": "john.doe@example.com",
  "PhoneNumber": "1234567890",
  "Address": "123 Main St",
  "City": "New York",
  "PostalCode": "10001",
  "StateId": 1,
  "CountryId": 1,
  "CountryCode": "+1"
}
```

### Example Response

<Response>

```json
{
  "status_code": 200,
  "message": "Customer created successfully",
  "data": {
    "CustomerId": "fe8bd3d5-9ebd-4722-bf44-23d88c250b34",
    "CustomerName": "John Doe",
    "Email": "john.doe@example.com",
    "PhoneNumber": "1234567890",
    "Address": "123 Main St",
    "City": "New York",
    "PostalCode": "10001",
    "StateId": 1,
    "CountryId": 1,
    "CountryCode": "+1",
    "GatewayCustomerId": "d0h93vn0i47dbksokv70",
    "createdAt": "2025-05-13T00:27:43.000Z"
  }
}
```

</Response>

## Customer List

This endpoint retrieves a paginated list of all customers associated with your account, with options for sorting and searching.

<Endpoint method="POST" url="https://dev-api.auxvault.net/api/v1/public/customers">

<ParamField type="header" name="Authorization" required>
  Your API key for authentication.
</ParamField>

<ParamField body="search" type="string">
  A search term to filter customers by name or email.
</ParamField>

<ParamField body="sort" type="string">
  The sorting order for the results. Can be `asc` or `desc`.
</ParamField>

<ParamField body="page" type="integer">
  The page number for pagination.
</ParamField>

<ParamField body="perPage" type="integer">
  The number of results to return per page.
</ParamField>

</Endpoint>

### Example Request

```json
{
  "search": "John",
  "sort": "asc",
  "page": 1,
  "perPage": 10
}
```

### Example Response

<Response>

```json
{
  "status_code": 200,
  "message": "Customer found successfully",
  "data": [
    {
      "CustomerId": "fe8bd3d5-9ebd-4722-bf44-23d88c250b34",
      "CustomerName": "xxxxxxxxxxxxx",
      "Address": "",
      "City": "",
      "PostalCode": "54321",
      "StateId": null,
      "CountryId": 1,
      "PhoneNumber": "8976732233",
      "Email": "xxxxxxxxxxxx@gmail.com",
      "GatewayCustomerId": "d0h93vn0i47dbksokv70",
      "CountryCode": "+1",
      "createdAt": "2025-05-13T00:27:43.000Z"
    }
  ],

```

</Response>
