---
title: 'Token Transaction API'
description: 'Process transactions using previously tokenized payment methods'
---

This endpoint enables you to process transactions using previously tokenized payment methods securely.

## API Endpoint
`https://dev-api.auxvault.net/api/v1/public/token-transaction`

## Request Parameters
The request must be a POST request with the following parameters:

| Parameter            | Data Type | Required | Description                    |
|---------------------|-----------|----------|--------------------------------|
| Amount              | Decimal   | Yes      | The amount of the recurring    |
| ConvenienceFeeActive| Boolean   | Yes      | For adding the fee             |
| Description         | String    | No       | Purpose of transaction         |
| GatewayCustomerId   | String    | Yes      | For internal purpose           |
| ReferenceNo         | String    | No       | For transaction reference      |
| TokenId             | String    | Yes      | For internal purpose           |

## Example Request
```json
{
  "Amount": "10",
  "ConvenienceFeeActive": true,
  "Description": "test",
  "GatewayCustomerId": "d0h93vn0i47dbksokv70",
  "ReferenceNo": "1123",
  "TokenId": "d0h93vn0i47dbksokv70"
}
```

## Response
The response will contain the status, message, and data of the transaction.

| Parameter | Data Type | Description                              |
|-----------|-----------|------------------------------------------|
| status    | String    | The status of the transaction            |
| message   | String    | A message describing the transaction status |
| data      | Object    | An object containing details of the transaction |

**Example Response**:
```json
{
  "status": "success",
  "status_code": 200,
  "message": "Transaction Processed Successfully",
  "data": {
    "TransactionId": "d0hkp170i47dbksons00",
    "BaseAmount": 10,
    "Amount": 10,
    "CardNumber": "4242",
    "BillingEmail": "xxxxxxxxxxxx@gmail.com",
    "BillingCustomerName": "xxxxxxxxxxxxx",
    "BillingAddress": "",
    "BillingCity": "",
    "BillingPostalCode": "54321",
    "BillingPhoneNumber": "8976732233",
    "ExpiryDate": "1225"
  }
}
```

## Error Codes
If an error occurs, the response will contain an error code and a message.

| Error Code | Description               |
|------------|---------------------------|
| 400        | Bad Request               |
| 401        | Unauthorized              |
| 403        | Forbidden                 |
| 404        | Not Found                 |
| 500        | Internal Server Error     |

## Security
To access this API endpoint, you will need to provide an API key in the header:

```javascript
var request = { 
  method: "post", 
  url: "https://dev-api.auxvault.net/api/v1/public/token-transaction", 
  headers: { 
    "Content-Type": "application/json", 
    "authorization": "APIKEY"
  }, 
  data: data
}
```
