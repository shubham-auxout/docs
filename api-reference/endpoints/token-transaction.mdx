---
title: 'Token Transaction API'
description: 'Process transactions using previously tokenized payment methods'
---

This endpoint enables you to process transactions using a previously generated token, providing a secure way to handle recurring or future payments without storing sensitive card details.

<Endpoint method="POST" url="https://dev-api.auxvault.net/api/v1/public/token-transaction">

<ParamField type="header" name="Authorization" required>
  Your API key for authentication.
</ParamField>

<ParamField body="TokenId" type="string" required>
  The unique token representing the customer's payment method.
</ParamField>

<ParamField body="GatewayCustomerId" type="string" required>
  The customer identifier from the payment gateway.
</ParamField>

<ParamField body="Amount" type="decimal" required>
  The amount to be charged for the transaction.
</ParamField>

<ParamField body="ConvenienceFeeActive" type="boolean" required>
  Set to `true` to apply a convenience fee to the transaction.
</ParamField>

<ParamField body="Description" type="string">
  An optional description for the transaction.
</ParamField>

<ParamField body="ReferenceNo" type="string">
  An optional reference number for your internal tracking.
</ParamField>

</Endpoint>

## Example Request

```json
{
  "TokenId": "d0h93vn0i47dbksokv70",
  "GatewayCustomerId": "d0h93vn0i47dbksokv70",
  "Amount": "10.00",
  "ConvenienceFeeActive": true,
  "Description": "Monthly Subscription",
  "ReferenceNo": "INV-12345"
}
```
