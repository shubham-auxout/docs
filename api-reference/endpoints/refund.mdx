---
title: 'Refund Transaction API'
description: 'Process refunds for settled transactions'
---

This endpoint enables you to process a full or partial refund for a previously settled transaction.

<Endpoint method="POST" url="https://dev-api.auxvault.net/api/v1/public/refund">

<ParamField type="header" name="Authorization" required>
  Your API key for authentication.
</ParamField>

<ParamField body="TransactionId" type="string" required>
  The unique identifier of the transaction to be refunded.
</ParamField>

<ParamField body="TransactionFor" type="string" required>
  The type of refund. Can be `FullRefund` or `PartialRefund`.
</ParamField>

<ParamField body="Amount" type="string" required>
  The amount to be refunded. For a `PartialRefund`, this must be less than or equal to the original transaction amount.
</ParamField>

<ParamField body="Reason" type="string">
  An optional reason for the refund.
</ParamField>

</Endpoint>

## Example Request

```json
{ 
  "TransactionId": "chcfbh46lyu7653qe",
  "TransactionFor": "PartialRefund",
  "Amount": "10.00",
  "Reason": "Customer requested a partial refund."
}
```
