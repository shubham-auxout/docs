---
title: 'Refund Transaction API'
description: 'Process refunds for settled transactions'
---

This endpoint enables you to process a refund for a transaction with the provided details.

## API Endpoint
`https://dev-api.auxvault.net/api/v1/public/refund`

## Request Parameters
The request must be a POST request with the following parameters:

| Parameter       | Data Type | Required | Description                   |
|----------------|-----------|----------|-------------------------------|
| Amount         | String    | Yes      | The amount of the transaction |
| Reason         | String    | No       | The reason for refund         |
| TransactionFor | String    | Yes      | FullRefund / PartialRefund    |
| TransactionId  | String    | Yes      | The transaction id            |

## Example Request
```json
{ 
  "Amount": "100",
  "Reason": "", 
  "TransactionFor": "Refund", 
  "TransactionId": "chcfbh46lyu7653qe"
}
```

## Response
The response will contain the status, message, and data of the transaction.

| Parameter | Data Type | Description                              |
|-----------|-----------|------------------------------------------|
| status    | String    | The status of the transaction            |
| message   | String    | A message describing the transaction status |
| data      | Object    | An object containing details of the transaction |

## Error Codes
If an error occurs, the response will contain an error code and a message.

| Error Code | Description               |
|------------|---------------------------|
| 400        | Bad Request               |
| 401        | Unauthorized              |
| 403        | Forbidden                 |
| 404        | Not Found                 |
| 500        | Internal Server Error     |

## Security
To access this API endpoint, you will need to provide an API key in the header:

```javascript
var request = { 
  method: "post", 
  url: "https://dev-api.auxvault.net/api/v1/public/refund", 
  headers: { 
    "Content-Type": "application/json", 
    "Authorization": "APIKEY"
  }, 
  data: data
}
```
