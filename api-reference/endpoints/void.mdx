---
title: 'Void Transaction API'
description: 'Void authorized transactions before they are settled'
---

This endpoint allows you to void (cancel) a previously processed transaction using the provided transaction details.

## API Endpoint
`https://dev-api.auxvault.net/api/v1/public/void`

**Production Endpoint**: `https://api.auxvault.net/api/v1/public/transaction`

## Request Parameters
The request must be a POST request with the following parameters:

| Parameter       | Data Type | Required | Description                   |
|----------------|-----------|----------|-------------------------------|
| Reason         | String    | No       | The reason for void           |
| TransactionFor | String    | Yes      | Void                          |
| TransactionId  | String    | Yes      | The transaction id            |

## Example Request
```json
{ 
  "Reason": "", 
  "TransactionFor": "Void", 
  "TransactionId": "chcfbh46lyu7653qe"
}
```

## Response
The response will contain the status, message, and data of the transaction.

| Parameter | Data Type | Description                              |
|-----------|-----------|------------------------------------------|
| status    | String    | The status of the transaction            |
| message   | String    | A message describing the transaction status |
| data      | Object    | An object containing details of the transaction |

**Example Response**:
```json
{
    "status": "success",
    "status_code": 200,
    "message": "Transaction Void Initiated Successfully",
    "data": {
        "TransactionId": "chcfbh46lr8uj3go8t30",
        "type": "Void"
    }
}
```

## Error Codes
If an error occurs, the response will contain an error code and a message.

| Error Code | Description               |
|------------|---------------------------|
| 400        | Bad Request               |
| 401        | Unauthorized              |
| 403        | Forbidden                 |
| 404        | Not Found                 |
| 500        | Internal Server Error     |

## Security
To access this API endpoint, you will need to provide an API key in the header:

```javascript
var request = { 
  method: "post", 
  url: "https://dev-api.auxvault.net/api/v1/public/void", 
  headers: { 
    "Content-Type": "application/json", 
    "Authorization": "APIKEY"
  }, 
  data: data
}
```
