---
title: Install & Activate
description: Install the AuxPay WooCommerce plugin and activate it in WordPress.
---

<Steps>
<Step title="Upload the plugin">
Go to Plugins → Add New → Upload Plugin → select the ZIP → Install Now.
</Step>
<Step title="Activate">
After install, click Activate. Alternatively, place the plugin folder in `wp-content/plugins/` and activate from Plugins.
</Step>
<Step title="Verify WooCommerce">
Make sure WooCommerce is installed and active.
</Step>
</Steps>

<Callout>
For production sites, ensure you have a staging environment to test updates.
</Callout>
